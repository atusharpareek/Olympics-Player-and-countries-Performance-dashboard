# Olympics-Player-and-countries-Performance-dashboard

🏅 Olympics Data Analysis Dashboard
📊 Overview

This project presents an interactive Olympics Dashboard that analyzes historical Olympic data up to 2016. It provides insights into athlete performance, medal distribution, country dominance, and trends across different Olympic Games.

The dashboard helps users explore how countries and athletes have performed over time using rich visualizations.

📁 Dataset
File: Olympics Dataset.xlsx
Contains:
Athlete name
Country
Sport
Event
Year
Medal type (Gold, Silver, Bronze)
Gender
